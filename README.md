# Weather Application

A simple and interactive weather application that allows users to search for a city and view current weather conditions such as temperature, humidity, and wind speed.

## Features
- Search weather by city name
- Displays temperature, humidity, and wind speed
- Dynamic weather icons based on weather conditions
- Error handling for invalid city names
- Clean and responsive user interface

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeather API

## How It Works
- Users enter a city name in the search bar
- JavaScript fetches real-time weather data from the OpenWeather API
- The UI updates dynamically with the retrieved data
- An error message is displayed if the city name is invalid
